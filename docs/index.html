Message Counter

Count your sent messages!

Message Counter is an Android app that counts the number of SMS sent in a cycle and notifies you when you have reached the limit.

Features

  • Track the number of SMS messages in inbox, sent and total.
  • Track number of SMS sent per each cycle
  • Track number of SMS sent each week
  • Notification when SMS limit per cycle is reached
  • List of top contacts who have sent you messages
  • Material Design
  • Widgets
  • Multiple language support
  • No Advertisements
