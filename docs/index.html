"An Android app that counts the number of SMS sent in a cycle with notification".

Message Counter has been downloaded more than 80,000 times from the Google Play Store and used in more than 10 countries. It is designed with Google's Material design principles in mind.

The app is both Free and Open Source.

The latest version of Message Counter is Version 3.2 and is available from the Google Play Store.

Features

  • Track the number of SMS messages in inbox, sent and total.
  • Track number of SMS sent per each cycle
  • Track number of SMS sent each week
  • Notification when SMS limit per cycle is reached
  • List of top contacts who have sent you messages
  • Material Design
  • Widgets
  • Multi language support (English, Spanish, French, Portuguese)
  • No Advertisements
